# iKhaya Health - User Guide

## Getting Started

### Installation
1. Download the app for your device (Android/iOS/Windows/macOS/Linux)
2. Install and launch the application
3. Log in with your clinic credentials

### First Time Setup
1. Configure clinic information
2. Set up user accounts for staff
3. Test offline functionality
4. Perform initial sync

## Daily Operations

### Registering a New Patient
1. Click "Add Patient" button
2. Fill in patient details
3. Take or upload patient photo
4. Save record (works offline)

### Viewing Patient Records
1. Use search bar to find patient
2. Click patient name to view full record
3. Review visit history

### Recording a Visit
1. Open patient record
2. Click "New Visit"
3. Record vitals, symptoms, diagnosis
4. Prescribe treatment
5. Save visit (works offline)

### Syncing Data
- Automatic: App syncs when internet is available
- Manual: Click sync button in top right
- Status: Green = synced, Yellow = pending, Red = conflict

## Offline Mode
- All features work without internet
- Changes saved locally
- Automatic sync when online
- Conflict resolution if needed

## Troubleshooting

### Sync Issues
- Check internet connection
- Verify login credentials
- Contact clinic administrator

### Performance Issues
- Close unused apps
- Clear app cache
- Update to latest version
